【SORA KPI Webフォーム GitHubアップロード用パッケージ】

1. このフォルダ内のファイルをそのままGitHubリポジトリにアップロードしてください。
2. `config.js` は含まれていません。ローカル動作用にのみ保持してください。
3. `config_sample.js` を参考に、本番ではAPIキーをサーバー側に移行することをおすすめします。
4. VercelやNetlifyにデプロイする場合、環境変数（APIキー）は管理画面で登録可能です。
5. 地図や郵便番号補完はGitHub Pages上でも問題なく動作します。
